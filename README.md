# 智能学习助手 - 个性化学习资源生成系统

## 项目简介

本项目是一个基于多智能体协同的个性化学习资源生成系统，旨在为学生提供智能化、定制化的学习支持。系统通过对话式采集学生画像，结合RAG知识库检索，利用多智能体协作生成课程讲义、练习题、思维导图、PPT等学习资源，并制定个性化学习路线。

### 赛题背景

随着教育信息化的发展，个性化学习已成为教育领域的重要研究方向。本系统针对《计算机网络》课程，利用人工智能技术实现：
- 学生画像智能采集与分析
- 多模态学习资源自动生成
- 个性化学习路径规划
- 基于知识库的内容检索与生成

## 部署步骤

### 1. 安装依赖

```bash
cd project_root
pip install -r requirements.txt
```

### 2. 密钥配置

在项目根目录创建 `.env` 文件，配置讯飞API密钥：

```env
# 讯飞星火大模型配置
SPARK_APP_ID=your_spark_app_id
SPARK_API_KEY=your_spark_api_key
SPARK_API_SECRET=your_spark_api_secret

# 讯飞多模态配置
MULTIMODAL_APP_ID=your_multimodal_app_id
MULTIMODAL_API_KEY=your_multimodal_api_key
MULTIMODAL_API_SECRET=your_multimodal_api_secret

# 可选配置
APT_MIRROR=https://mirrors.tuna.tsinghua.edu.cn/ubuntu/
CREWAI_LOGGING=true
```

### 3. 知识库导入

将《计算机网络》课程资料PDF文件放入 `data/pdfs/` 目录：

```
data/
└── pdfs/
    ├── 计算机网络-第一章.pdf
    ├── 计算机网络-第二章.pdf
    └── ...
```

系统启动时会自动扫描该目录并构建向量索引。

### 4. 启动命令

#### 方式一：命令行运行（后台任务）

```bash
python main.py
```

#### 方式二：Web界面运行

```bash
streamlit run app.py
```

启动后访问：http://localhost:8501

## 功能演示操作步骤

### 演示步骤一：学生画像采集

1. 启动Web界面：`streamlit run app.py`
2. 点击左侧"学生画像采集"
3. 输入学号和姓名，点击"开始采集"
4. 与智能助手进行多轮对话，回答关于学习风格、薄弱点等问题
5. 点击"生成画像"，查看6维画像卡片

### 演示步骤二：学习资源生成

1. 点击"前往学习需求"（需先完成画像采集）
2. 输入学习目标，如："掌握计算机网络核心知识"
3. 输入重点知识点（每行一个）：
   - TCP三次握手
   - HTTP协议
   - IP地址与子网
4. 点击"开始生成"，等待多智能体执行
5. 查看执行日志和生成进度条
6. 点击"生成思维导图"和"生成PPT"创建多模态资源

### 演示步骤三：查看学习路线

1. 点击"查看学习路线"
2. 查看自动生成的个性化学习计划
3. 浏览历史学习资料记录
4. 查看资源统计面板

### 演示步骤四：命令行全流程

```bash
python main.py
```

系统将自动执行：
1. 初始化数据库和向量库
2. 模拟对话采集学生画像
3. 生成学习资源（讲义、练习题、学习计划）
4. 生成思维导图和PPT
5. 保存所有资源到数据库

## 技术亮点

1. **多智能体协同**：基于CrewAI实现6个专业智能体的任务调度与并行执行
2. **RAG检索增强**：结合Chroma向量数据库实现精准的知识点检索，防止大模型幻觉
3. **多模态生成**：集成讯飞多模态接口，支持思维导图、PPT、视频等资源生成
4. **个性化画像**：6维度学生画像采集，动态更新薄弱知识点
5. **可视化界面**：Streamlit构建美观的Web交互界面

## 注意事项

1. 确保已申请讯飞API密钥并正确配置
2. 首次启动需要较长时间构建向量索引
3. 多模态文件生成需要联网调用讯飞接口
4. 建议使用虚拟环境运行项目
