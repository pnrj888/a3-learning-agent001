> **场景设定：** 一名大三学生（小王），准备考研数学一，第一次使用本系统
> **演示风格：** 自然流畅，突出AI智能感 + 个性化体验
> **屏幕比例：** 16:9 Web端

## 附：DEMO执行要点

| 项目 | 要求 |
|------|------|
| **总时长** | 3分钟（±10秒） |
| **操作员** | 1人（演示者兼任，提前熟悉点击位置） |
| **网络要求** | 需预加载/缓存部分内容，避免生成延迟过长 |
| **建议背景音乐** | 轻快Lo-fi学习风格，音量-20dB |
| **字体** | 屏幕字体至少14px，确保评委看得清 |
| **重点强调（语速放慢）** | ①画像生成时刻 ②拖拽重排 ③质检Agent纠错 ④学习报告雷达图对比 |
| **备选预案** | 若实时生成延迟 >3秒，准备录屏素材作为插片 |
